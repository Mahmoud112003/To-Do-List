# To-Do List Web Application

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**.  
This project allows users to manage their tasks efficiently with features like adding, deleting, completing tasks, a task counter, local storage, a clear all button, and drag & drop reordering.

---

## **Features**

- Add new tasks  
- Mark tasks as completed  
- Delete individual tasks  
- Clear all tasks  
- Task counter  
- Tasks saved in **LocalStorage** (persist after page refresh)  
- Add task by pressing **Enter** key
